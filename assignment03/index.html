Pusheen's Cheat Day

Inspirations Behind the Game


Pusheen's Cheat Day was largely inspired by the dinosaur game that people can play on the Google Chrome web browser when there is no wifi available. I've always personally enjoyed playing the dinosaur game even though it is very simple. In my opinion, it's the simplicity in the game that is able to captivate so many people. I also took inspiration from Pusheen the Cat as I wanted to add a cuter element to the game. It was easy to assign an 'objective' as well as an 'obstacle' in this game as Pusheen is known to be a lazy cat who enjoys desserts. Finding the graphics for this project proved to be the most difficult part as many of the images I wanted to use were not tranparent PNGs; therefore, I had to take the time and manually cut out each graphic. However, as painstaking as it was, I enjoyed being able to control what graphics I used and would want to continue making other games that test my creativity as well as my art skills.
